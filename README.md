# 球王之路

一个本地离线运行的纯文字足球球员生涯游戏。打开 `index.html` 即可游玩；在支持 Service Worker 的安全来源下，也可以安装成 PWA 离线网页。

## 存档

游戏使用浏览器 `localStorage` 本地存档。PWA 离线缓存只负责保存网页文件，不会替代或清空你的生涯档案。

档案页提供“导出存档”和“导入存档”。如果换手机、换浏览器、换网址来源，先导出 JSON，再在新环境导入即可继续。

同一个浏览器、同一个网址来源会读取同一份存档。下面这些情况会被浏览器当成不同来源：

- `file:///.../index.html`
- `http://127.0.0.1:4173/index.html`
- `http://192.168.x.x:4173/index.html`

换浏览器、清除网站数据、无痕模式、换来源地址，都可能看不到原来的档。

## 手机 PWA

PWA 离线安装需要安全来源：通常是 HTTPS，或者本机 `localhost`。直接用电脑局域网 IP，例如 `http://192.168.1.23:4173`，多数手机浏览器不会允许注册离线 Service Worker。

可选方案：

- 临时同 Wi-Fi 玩：电脑运行 `python -m http.server 4173 --bind 0.0.0.0`，手机打开电脑 IP。这样能玩，但不保证 PWA 离线安装。
- 真正手机离线 PWA：把项目文件放到手机，用手机本地服务器 App 提供 `http://localhost:端口`，在手机浏览器打开后添加到主屏幕。
- HTTPS 方式：用你自己的 HTTPS 静态托管或内网 HTTPS 服务打开一次，再添加到主屏幕。

## 已有机制

- 16 岁球员开局：姓名、国籍、位置、随机豪门青训或小球队一线队。
- 豪门青训开局：青训合约即将到期，经纪人会带来母队一线队、母队梯队续约、其他豪门梯队挖角等方案。
- 严格模拟：比赛、训练、能力成长、身价、薪水、转会、国家队征召都由游戏规则结算。
- 比赛：可以文字播报，也可以跳过直接看赛果。
- 经纪人：转会兴趣、青训挖角、续约、报价、推荐策略都会通过经纪人呈现。
- 转会窗：夏窗和冬窗会生成俱乐部买卖人、报价和流言。
- 积分榜：当前联赛会自动模拟，显示欧战、降级或升级区。
- 发布会与社交舆论：回答会影响主帅、董事会、球迷、队友和经纪人态度。
- 手机端适配：竖屏下按钮、积分榜和日志都能滚动操作。
